## States

- Draft: captured, not reviewed.
- Approved: accepted for implementation.
- Implemented: merged into `main`.
- Verified: acceptance tests pass.
- Released: included in tagged release.

- Requirements: `docs/product/v1/requirements.md`

- Breaking requirement changes require an ADR.
